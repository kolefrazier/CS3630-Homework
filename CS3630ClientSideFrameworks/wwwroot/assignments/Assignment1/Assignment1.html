How to Play

An object will be randomly picked - Rock, Paper or Scissors
You will be given a random condition to meet - Win, Lose, Tie

Your goal is to select the object that matches the condition for the random object.

Pick correctly, earn some points. Pick incorrectly or fail to pick before the timer runs out, lose double points.

Examples

Given Object is Paper, given Condition is Win. You need to pick the item that makes you win against Paper, which is scissors!

Given Object is Rock, given Condition is Lose. You need to pick the item that makes you lose against Rock, which is scissors!

Start with 20 points. You lose if you run out of time or lose all your points.
